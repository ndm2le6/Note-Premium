MỤC TIÊU PREMIUM - APK OFFLINE 100% BUILD BẰNG GITHUB ACTIONS

Không cần AndroidIDE build trên điện thoại.

Cách làm:
1. Giải nén ZIP này.
2. Tạo repository mới trên GitHub.
3. Upload toàn bộ nội dung trong thư mục MucTieuPremiumOffline_GitHubActions lên repo.
4. Vào tab Actions.
5. Chọn workflow "Build Offline APK".
6. Bấm "Run workflow".
7. Chờ build xong.
8. Mở run mới nhất, kéo xuống Artifacts.
9. Tải "MucTieuPremiumOffline-debug-apk".
10. Giải nén, lấy app-debug.apk và cài.

App chạy offline vì HTML nằm trong app/src/main/assets/index.html.
